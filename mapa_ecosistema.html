Respuesta rápida a las dudas típicas

¿Dónde está el MCP MedicoPC?Documents\doctorpc\remote-diagnostic-mcp\
¿Cómo actualizo el MCP?Tocar src\*.ts → npm run build → reiniciar Claude Desktop
¿Dónde veo mis emails de negocio?mail.zoho.eu (hola@medicopc.es)
¿Cómo cambio el DNS del dominio?dondominio.com → Dominios → medicopc.es → Parking & Zona DNS
¿Dónde está el manual del maletín?Hub → tab Docs → Manual del maletín
¿Qué USB llevo a la visita?🟢 MASTER siempre · 🔴 ARSENAL si sospechas "no arranca"
¿Dónde guardo el backup del cliente?💾 Seagate 4TB → MedicoPC_Backups_Clientes\[cliente]\

¿Cuál es la URL del MCP Personal?mcp-personal.medicopc.workers.dev/sse
¿Dónde apunto una idea del negocio?Claude Desktop → "apunta_idea" (MCP Personal)
